Application Chat (Java Networking) 
-------------------------------------------------------------------------------------
Cette application Client/Server qui permet à deux utilisateurs de
communiquer 
Exemple :
 ? Le première utilisateur (Client A) envoie un message texte puisse met en attente.
? De son côté, le deuxième utilisateur (Client B) reçoive le message et renvoie à
l'émetteur (Client A) un autre message puis se met en attente.
? Le première utilisateur (Client A) reste en attente jusqu’à ce qu'il reçoive le message
de deuxième utilisateur pour envoyer un nouveau message et ainsi de suite……
